چت‌های قبلی

درباره BK-AI

BK-AI یک پلتفرم چت هوش مصنوعی پیشرفته است که از مدل‌های مختلف هوش مصنوعی برای ارائه پاسخ‌های دقیق و سریع استفاده می‌کند. این پلتفرم توسط مهدی کلیدری توسعه یافته و هدف آن ارائه تجربه‌ای کاربرپسند و کارآمد برای کاربران فارسی زبان است.

ویژگی‌های کلیدی

  • پشتیبانی از چندین مدل هوش مصنوعی (GPT-4o و Gemini-2.5-Pro)
  • ذخیره‌سازی تاریخچه چت‌ها
  • رابط کاربری زیبا و واکنش‌گرا
  • پشتیبانی از تم تیره و روشن
  • امکان کپی و ارزیابی پاسخ‌ها
  • طراحی شده برای کاربران فارسی زبان

راهنمای استفاده

برای استفاده از BK-AI مراحل زیر را دنبال کنید:

چگونه چت کنیم

  1. مدل هوش مصنوعی مورد نظر خود را از لیست انتخاب کنید
  2. در کادر متن، پیام خود را تایپ کنید
  3. دکمه ارسال را بزنید یا Enter را فشار دهید
  4. پاسخ هوش مصنوعی پس از چند لحظه نمایش داده می‌شود

مدیریت چت‌ها

  • برای شروع یک چت جدید، روی دکمه "چت جدید" کلیک کنید
  • برای مشاهده چت‌های قبلی، روی دکمه "چت‌های قبلی" کلیک کنید
  • برای کپی کردن پاسخ، روی دکمه "کپی" در زیر پاسخ کلیک کنید
  • برای ارزیابی پاسخ، از دکمه‌های Like و Dislike استفاده کنید
